## Features

- **Frontend:** React, HTML, CSS  
- **State Management:** React Hooks (`useState`, `useEffect`)  
- **Real-Time Updates:** Polling / WebSockets  
- **Deployment:** Vercel, GitHub Pages, or Netlify
